# IndigoVersion
Version system for C/C++ programs.

## Usage

Command: `IndigoVersion.exe <directory> <product_name> <command>`
  * directory: Where the header is to be generated
  * command: The product name, and what the header will be named
    * Increase
    * Decrease
    * Reset
